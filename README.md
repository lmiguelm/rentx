## 💻 Como usar

- **_Clone o projeto:_** <br>
  ` git clone https://github.com/lmiguelm/rentx`

- **_Entre na pasta do projeto:_** <br>
  `cd rentx`

- **_Baixe as dependências:_** <br>
  ` yarn || npm install`

- **_Rode o app:_** <br>
  `yarn run android || npm run android`
  <br>
  `yarn run ios || npm run ios`
